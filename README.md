# leaderboard
>The leaderboard website displays scores submitted by different players. It also allows you to submit your score. All data is preserved thanks to the external Leaderboard API service.

## Built With

- HTML
- CSS
- Javascript

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- Use `git clone` to clone the repository using the HTTPS link into your local machine

### Setup

- `cd` into your local repository

### Install

- `npm install` - Installs the necessary dependencies

### Usage

- Open the index file and enjoy

### Deployment

- [Live Demo Link](https://zainab-alzubair.github.io/leaderboard/dist/)
